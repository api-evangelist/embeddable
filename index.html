Embeddable

An index and topic collection covering embeddable widgets, JavaScript SDKs, iframe-able experiences, and embedded analytics. Embeddable platforms allow product teams to drop pre-built UI components, payment flows, chat widgets, media players, forms, maps, dashboards, and authentication experiences directly into a host site or application via a script tag, iframe, or JS SDK. This collection includes payment elements like Stripe Elements and Stripe Checkout, financial linking like Plaid Link, mapping like Mapbox GL JS and Google Maps Embed, media players from YouTube, Vimeo, and Wistia, chat widgets from Intercom, Drift, Crisp, and Tawk-style services, scheduling widgets like Calendly, forms from HubSpot, Typeform, JotForm and Gravity Forms, identity via Auth0 Lock, chat SDKs from Stream and Sendbird, and embedded analytics from Looker, Apache Superset, Sigma, Cube, Mode, Metabase, Qlik Mashups, SAP BI Tools, Power BI, and Tableau.

extensionCommon Features

extensionDrop-in UI Components

Embeddable platforms ship pre-built, brandable UI components that drop into a host page via a script tag, web component, or framework binding without requiring developers to build the experience from scratch.

extensionIframe and Script-Tag Distribution

Most embeddable widgets are delivered as either a hosted iframe or a small JavaScript bootstrap that injects an isolated DOM subtree into the host site, insulating it from CSS and JS conflicts.

extensionTokenized and Scoped Authentication

Embeds use short-lived, server-issued tokens (publishable keys, link tokens, JWTs, session tokens) so the host site can render the widget without exposing long-lived provider credentials in the browser.

extensionTheme and Style Customization

Embeds expose theming hooks - color tokens, fonts, CSS variables, layout options - so the embedded experience matches the host product's visual identity.

extensionPostmessage and Callback Event Model

Embedded widgets communicate with the host page through postMessage events, callback functions, or SDK event listeners covering lifecycle states like ready, success, error, and dismiss.

extensionEmbedded Analytics and Dashboards

BI vendors like Looker, Sigma, Cube, Mode, Apache Superset, Metabase, Power BI, and Tableau ship embed SDKs and signed iframe URLs so customers can drop full dashboards into their own SaaS products.

extensionPCI, KYC, and Compliance Offloading

Embeds for payments, identity, and financial data (Stripe Elements, Plaid Link, Auth0 Lock) keep sensitive data inside the provider's frame so the host product stays out of PCI, KYC, and compliance scope.

extensionMobile SDK and Web Parity

Most embeddable platforms ship matching iOS, Android, and React Native SDKs alongside the JS widget so the same experience can be embedded across web and mobile apps.

task_altUse Cases

task_altAccepting Payments Without PCI Scope

Drop Stripe Elements, Stripe Checkout, Braintree Drop-in, Adyen Components, or Square Web Payments SDK into a checkout flow to capture cards without the host server ever touching raw PAN data.

task_altLinking Bank Accounts and Financial Data

Embed Plaid Link in a fintech onboarding flow so users can authenticate with their bank, link accounts, and grant data access without leaving the host application.

task_altEmbedding Maps and Location Pickers

Drop Mapbox GL JS or the Google Maps Embed into a marketplace, real estate, or logistics product to render interactive maps, search by address, and place markers without building map tiles in-house.

task_altAdding Live Chat and Support Widgets

Embed Intercom Messenger, Drift, Crisp, Olark, LiveChat, Help Scout Beacon, Zendesk Web Widget, or Freshdesk Messaging to add support and sales chat to any site through a single script tag.

task_altInline Scheduling and Booking

Embed Calendly or Cal.com inline widgets into marketing sites and SaaS dashboards so prospects and customers can book meetings without bouncing to a third-party scheduler.

task_altEmbedded Forms and Lead Capture

Drop HubSpot Forms, Typeform, JotForm, Gravity Forms, or Formspree into landing pages and product surfaces to capture leads, feedback, and onboarding data without backend work.

task_altEmbedded BI Dashboards in SaaS

SaaS vendors embed Looker, Sigma, Cube, Mode, Apache Superset, Metabase, Knowi, Qlik Mashups, Power BI, SAP BI Tools, or Tableau to give customers in-app dashboards branded as part of the host product.

task_altLogin, Signup, and Identity as a Widget

Embed Auth0 Lock or hosted login pages to give the host product a full identity experience - email/password, social, MFA, passwordless - without building auth screens.
