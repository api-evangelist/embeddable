Embeddable

Embeddable is focused on tools that provide copy and paste code, that syndicates and distributes data and content around the web. This site focuses on using embeddable tools in the name of APIs, but not always directly using APIs for driving the embeddable tools.

Commonly known embeddable tools are the Twitter button, Facebook like and others buttons, badges and widgets. Usually embeddable tools are JavaScript which lends itself to the largest potential being a client side scripting language, but embeddable tools can be as simple as iFrame, HTML + CSS or even a simple image.

The goal of this project is to study how approaches to embeddable tools can be applied as part of an API ecosystem strategy. There are two sides of the embeddable coin, one for API providers and the other for ApI consumers.

Embeddable tools are not just for developers, they can be copy / pasted and reverse engineered by non-developers, making embeddable tools very powerful among the power user group.

While Twitter's success is often claimed to be owned by the developer community, it was the ability to share, retweet and syndicate API driven, Twitter resources across the Internet.

A successful embeddable API strategy starts here.
