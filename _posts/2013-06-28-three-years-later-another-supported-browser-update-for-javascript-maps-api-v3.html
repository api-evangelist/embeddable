The times, they are a changin’.Back in 2010, when we last updated the supported browsers for JavaScript Maps API v3, Android was just starting to take off, Chrome was still very nascent and in beta on OS X and Linux, and Internet Explorer 9 had just been released.Fast-forward three years to today: it’s clear that Android is here to stay, Chrome, now one of the most popular browsers in the world, is available on mobile devices, and we have a brand new version of Internet Explorer.So, it’s time for another update.The following browsers and operating systems are now officially supported by the JavaScript Maps API v3: Nothing changes based on this announcement: sites working in a now-unsupported browser will continue to function as before.
